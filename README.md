# 🚀 DuckCore AI – Code Explainer & Debugger

DuckCore AI is a full-stack web application that helps users understand, debug, and optimize their code using intelligent processing.

---

## 📌 🔥 Features

- ✨ Code Explanation (easy understanding)
- 🐛 Debugging Suggestions
- ⚡ Code Optimization
- 🔐 User Authentication (Login System)
- 🕘 Chat History Storage
- 💻 Multi-language Support
- 🎯 Clean and Interactive UI

---

## 🧠 💡 How It Works

1. User logs into the system  
2. Enters code in the editor  
3. Selects action (Explain / Debug / Optimize)  
4. Backend processes the request  
5. Result is displayed  
6. Data is stored in database  

---
